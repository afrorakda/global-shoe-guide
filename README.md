# #64 Global Shoe Guide

A hyper-minimalist kids' shoe size converter designed for global parents. It instantly synchronizes US, UK, EU, and CM (Metric) sizes, removing the friction of international shopping and second-hand gear hunting.

## Core Philosophy
* **1 Tool / 1 Action**: Select your known size from any unit, and all other international scales update instantly.
* **Global Accessibility**: Features a 4-way unit switcher (US/UK/EU/CM) to cater to parents worldwide.
* **Growth Tracking**: Includes a "Next Size Goal" feature to help parents plan for their child's rapid growth.

## Key Features
* **Multi-Unit Switcher**: Toggle between US, UK, EU, and CM start points to find your match in seconds.
* **Comprehensive Range**: Covers from early walkers (12cm) to junior sizes (24cm/US 6) to span the most confusing growth phases.
* **Inches Support**: Automatically displays measurements in inches for users accustomed to imperial rulers.
* **Safety First**: Clearly emphasizes brand-specific size variances above the primary action buttons to manage user expectations.

## Design System
* **Background**: #fff
* **Main Accent**: #333
* **Border Radius**: 15px
* **Navigation**: High-contrast tab system for effortless unit switching.
* **iOS Optimization**: Enhanced tap targets and crisp rendering for mobile browsers.

## Privacy & Performance
* **No Tracking / No Ads / No Login**
* **Browser-only**: No data storage or external requests.
* **Ultra-Lightweight**: Minimal code for instant loading on any mobile device.
